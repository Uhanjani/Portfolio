Smart Object Detection & Home Appliances Control

IoT & AI-Based Security System

Using Raspberry Pi, TensorFlow, OpenCV

This project integrates AI-powered object detection and IoT-based home automation to enhance security and control home appliances remotely. It uses deep learning, IoT devices, and cloud computing to provide a real-time monitoring system.

Key Features:

  • AI-Powered Object Detection: Uses CNN + LSTM for accurate object recognition.
  • Home Appliance Control: Remotely switch ON/OFF devices via mobile app or automation.
  • Real-Time Alerts: Sends email/SMS notifications when a suspicious object/person is detected.
  • Offline & Online Support: Works via GSM messages when Wi-Fi is unavailable.
  • Cloud Integration: Stores data on AWS for secure access and remote control.
  • Smart Automation: Adjusts appliance settings based on detected activity.

How It Works:

  1. Object Detection: Camera connected to Raspberry Pi analyzes live footage.
  2. AI Processing: TensorFlow and OpenCV detect people or objects.
  3. Alert & Action: If a person is detected, an alert is sent, and appliances can be controlled.
  4. Cloud & Mobile App: Users monitor live feeds and control appliances via a mobile app.
  5. Offline Support: Uses GSM SMS commands when the internet is unavailable.

Future Enhancements:

  • Facial Recognition for Security
  • Voice Commands for Home Control
  • AI-based Predictive Appliance Scheduling
  • Integration with Smart Assistants (Google Home, Alexa)
